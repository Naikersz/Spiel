# Информация о кадрах для idle анимаций в Pygame

## Структура спрайт-шитов в Pygame

В Pygame для idle анимации используются **те же строки (rows), что и для walk**, но **только первый кадр (frame 0)**.

### Правила из `character.py` (строки 52-64):
```python
elif animation_type == "walk":
    row = self.WALK_DIRECTIONS.get(direction, 10)
    max_frames = 9
else:  # idle и другие
    row = self.WALK_DIRECTIONS.get(direction, 10)
    max_frames = 1  # Только 1 кадр для idle
```

### WALK_DIRECTIONS (строки 7):
```python
WALK_DIRECTIONS = {"up": 8, "left": 9, "down": 10, "right": 11}
```

## Координаты кадров в Pygame (для всех слоев одежды)
